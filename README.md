# Resume Craft 🚀

**Resume Craft** is a modern, high-performance resume builder application built with Flutter. It leverages the power of `flutter_hooks` and `riverpod` to provide a seamless, real-time resume creation experience across multiple platforms.

---

## ✨ Features

- **Real-time Preview**: See your resume take shape as you type.
- **Multiple Export Formats**: Export your polished resume to professional **PDF** or clean **Markdown**.
- **Modern UI/UX**: Features a vibrant indigo and emerald theme designed for clarity and visual appeal.
- **Native Experience**: Native file system integration on Android and Linux for easy saving.
- **Responsive Design**: Adapts beautifully to mobile, tablet, and desktop screens.
- **State Persistence**: Uses advanced state management for a smooth user journey.

## 🛠️ Built With

- **Flutter SDK**: `^3.10.8`
- **Architecture**: `hooks_riverpod` + `flutter_hooks`
- **Typography**: `Google Fonts (Outfit)`
- **PDF Generation**: `pdf` and `printing`
- **Rich Text**: `flutter_markdown`
- **Animations**: `lottie` and `animated_splash_screen`

## 🚀 Getting Started

### Prerequisites

Ensure you have the Flutter SDK installed on your machine. You can follow the [official Flutter installation guide](https://docs.flutter.dev/get-started/install).

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/xetri/resume_craft.git
   ```

2. Navigate to the project directory:
   ```bash
   cd resume_craft
   ```

3. Install dependencies:
   ```bash
   flutter pub get
   ```

4. Run the application:
   ```bash
   flutter run
   ```
